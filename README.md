# Pepe Graf
Это набор нод для obsidian который был создан на основе подписок и подписчиков игроков pepelend на NameMC
Скрипт на сбор страниц и их обработку наслопил за 1,5 дня если хотите с ним побаловаться он лежит в релизах. 
Для того чтобы ознакомиться с древом графов разархевируте obsidian_notes.zip в папку хранилища obsidian которую вы создали при первом запуске obsidian .
