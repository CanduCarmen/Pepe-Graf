# Pepe Graf
Это набор нод для obsidian который был создан на основе подписок и подписчиков игроков pepelend на NameMC по теории рукопожатий.
Скрипт на сбор страниц и их обработку наслопил за 1,5 дня если хотите с ним побаловаться он лежит в релизах. 
В графе указаны примерно все +- активные пепелендовци и горсть других косвенно связанных с ними игроков.
В релизном наборе нод примерно 6100 игроков и он может чутьб чутьб подлагивать.У каждой ноды игрока в графе есть описание с сcылкой на профиль NameMC. 
Из за кривости скачивания профилей игроков у некоторых игроков может не быть привью скина и другой информации, если игрок менял ник то его профиль тоже будет указан неточно из за специфики NameMC.
Если хотите ознакомиться с древом графов разархивируйте obsidian_notes.zip в папку хранилища obsidian которую вы создали при первом запуске obsidian
